🎮 Installation Le Lapin Blanc

📱 Method 1: Serveur Local Simple

1. Décompressez le dossier sur votre ordinateur

2. Ouvrez un terminal dans le dossier

3. Tapez: python -m http.server 8000

4. Ouvrez: http://localhost:8000

🌐 Method 2: Ouverture Direct

1. Double-cliquez sur index.html

2. L'application s'ouvre dans votre navigateur

3. Cliquez "Installer" pour l'ajouter à votre téléphone

📲 Installation Mobile

Android: Menu (⋮) → "Ajouter à l'écran d'accueil"

iPhone: Partager → "Sur l'écran d'accueil"

Desktop: Icône d'installation dans la barre d'adresse
